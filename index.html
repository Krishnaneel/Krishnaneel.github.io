    P R O B L E M

    As an employee management system, I need to create a web application with three screens.

    The second screen is a list of all registered employees, each with a picture randomly generated from (https://picsum.photos/) , job-id, name, and bioJoke. The list can be sorted by name, and clicking on an employee's card will lead to the third screen.

    The first screen is a form that allows users to register new employees by providing job-id, name, address, department, salary, and a bioJoke. The job-id will be auto-generated and must be an alphanumeric string with 6 characters. The name field must contain only alphabets, and the salary field must be a number. All fields are required. The bioJoke field will be hidden, and its value will be fetched from an API (https://icanhazdadjoke.com/api) that provides a random joke.

    The third screen shows all details of the selected employee, either as a modal or a pop-up.
